# WorkFlowy Forwarder Plus

WorkFlowy 增强工具，提供任务管理、提醒、标签筛选等功能。

## 功能说明

1. 基础配置
   - 目标节点ID
   - 跟踪标签
   - 刷新间隔

2. 数据存储
   - 使用 localStorage
   - 支持配置导入导出
   - 自动保存

3. 界面交互
   - 配置面板
   - 实时预览
   - 错误提示
